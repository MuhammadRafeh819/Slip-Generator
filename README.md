# Slip-Generator
The **Enterprise Pay Slip Generator** is a sophisticated web application that allows HR professionals and administrators to generate professional employee payslips instantly. The system features automatic bonus calculation based on salary brackets, real-time preview updates, persistent employee IDs, and print-optimized A4 output.

🎯 Key Features

- **⚡ Real-time Generation** - Payslip updates dynamically as you type or click Generate
- **🎯 Smart Bonus Engine** - Automatic bonus calculation with tiered rates (10% - 30%)
- **🖨️ Print-Ready Output** - Optimized CSS for A4 paper printing with print-specific styling
- **🔑 Persistent Employee IDs** - Unique IDs that remain stable within a session (e.g., "JD-12345")
- **📱 Fully Responsive** - Works seamlessly on desktop, tablet, and mobile devices
- **🎨 Modern UI** - Clean, gradient-free design with smooth animations
- **📊 Visual Bonus Policy** - Clear reference table showing all bonus tiers
- **🔄 Reset Functionality** - One-click form reset with fresh employee ID generation

## 🎮 Bonus Structure

| Salary Range (USD) | Bonus Rate | Performance Tier |
|-------------------|------------|------------------|
| $50,000 - $60,000 | 10% | Bronze |
| $60,001 - $70,000 | 15% | Silver |
| $70,001 - $80,000 | 20% | Gold |
| $80,001 - $90,000 | 25% | Platinum |
| $90,001 - $100,000 | 30% | Diamond |
| Below $50,000 or Above $100,000 | 0% | Standard |

## 🚀 Getting Started

- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No server, database, or internet connection required (runs entirely client-side)
- Google Fonts (Inter) will load if internet is available
